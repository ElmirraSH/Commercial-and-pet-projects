# 🌍 International Pharmaceutical Registry Normalization

## 📌 О проекте

В рамках проекта выполнена унификация, очистка и стандартизация фармацевтических реестров нескольких стран с целью создания единого аналитически пригодного датасета.

Основная задача — привести разрозненные государственные реестры лекарственных средств к единой структуре и каноническому виду для дальнейшей аналитики и построения BI-дашбордов.

---

## 🌎 Используемые источники данных

В проекте использованы реестры:

- 🇺🇬 Uganda  
- 🇩🇿 Algeria  
- 🇸🇦 Saudi Arabia  

### Дополнительно

- Файл **Saudi Arabia CSV** был получен путём парсинга официального реестра.
- Файл **all_countries.csv**, использованный для стандартизации стран производства, был спарсен со страницы **Wikipedia**.
- Для обогащения данных использована классификация **WHO ATC-DDD** от **World Health Organization**.
- Для стандартизации лекарственных форм использовалась таксономия **FDA SPL** от **Food and Drug Administration**.

---

## 🎯 Цель проекта

Создать единый стандартизированный датасет, пригодный для:

- кросс-странового анализа  
- анализа доступности препаратов  
- построения аналитических дашбордов  

---

## 🛠 Основной ноутбук

📘 **pharma_registry_normalization.ipynb**

Это ключевой файл проекта. В нём реализованы:

### 1️⃣ Унификация схем

- Созданы словари сопоставления колонок для каждой страны  
- Данные приведены к единой структуре  

### 2️⃣ Очистка данных

- Удалены строки без критически важных полей  
- Обработаны пропуски и заглушки (`---`)  
- Приведены типы данных (`datetime`, `numeric`)  
- Удалены дубликаты  

### 3️⃣ Текстовая нормализация

- Приведение к нижнему регистру  
- Удаление лишней пунктуации  
- Нормализация пробелов  

### 4️⃣ Интеллектуальная стандартизация (Fuzzy Matching)

Использована библиотека **RapidFuzz** для:

- стандартизации лекарственных форм  
- стандартизации производителей и дистрибьюторов  
- нормализации стран производства  
- исправления написаний действующих веществ  

### 5️⃣ Обогащение ATC-кодами

- Использована иерархия **WHO ATC-DDD**  
- Выполнено прямое сопоставление уникальных веществ  
- Применён fuzzy-поиск для сопоставления  
- Сохранены неоднозначные случаи
